# Triton/EKS Deferred

Triton and AWS/EKS scaffolding remains in this repository under `terraform/`, `k8s/`, and AWS GitHub workflows.
These assets are preserved for future production-path work and are not being removed.
They are not the active development path right now.
Current focus is the local Docker baseline for router strategy experiments on developer laptops.
We will revisit the Triton/EKS path after local baseline experiments are complete.
We will also revisit once GPU access and deployment reliability are stable again.
